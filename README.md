## :rocket: Project setup

use [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

```
npm / yarn install 
```

### Compiles and hot-reloads for development
```
npm / yarn run serve
```

### Compiles and minifies for production
```
npm / yarn run build
```
